##Build Instructions
(First time only)

`rm -rf node_modules`

`npm install`

(after the first time)

`ionic add platform $platform_of_choice`

`ionic build $platform_of_choice`

`ionic run $platform_of_choice`

(for quick debugging)

`ionic serve`

(this just runs the app on your computer and lets you play with it in the browser)
